# **App Name**: Mirrored Beauty

## Core Features:

- Page Cloning: Replicate the exact layout and content from goa.mylookscare.com to the new subdomain.
- Menu Replication: Mirror the menu structure from the existing site to maintain consistency.
- Blog Management: Enable adding, editing, and removing blog posts directly through the admin panel.
- Blog Page: Integrate a fully functional blog page, designed for easy navigation and content discovery.
- SEO Optimization: Implement tools and settings to optimize each blog post for search engines.

## Style Guidelines:

- Primary color: Soft coral (#F08080) to evoke a sense of warmth and care.
- Background color: Pale peach (#FAE0D4), creating a subtle backdrop that complements the content.
- Accent color: Muted lavender (#D8BFD8) to highlight calls to action and important elements.
- Body font: 'PT Sans', a humanist sans-serif suitable for body text.
- Headline font: 'Playfair', a modern serif font suitable for headlines. Use 'PT Sans' for body if longer text is anticipated.
- Utilize clean and modern icons, maintaining consistency with the existing 'mylookscare.com' branding.
- Maintain a clean, user-friendly layout mirroring the 'goa.mylookscare.com' design for consistency and familiarity.
- Incorporate subtle animations and transitions to enhance user experience and provide visual feedback.
